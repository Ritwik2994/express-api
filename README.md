
# Express Api Application

It is just a boiler-plate application for production grade Node.js project.

## Packages

- Node.js
- Express
- winston
  
## Features

- Have a typescript in its veins
- Custom errors for dealing with all kinds of basic logical errors
- winston for managing logs
- Docker enabled

## Installation

Requirements:

- Node.js

Installation Steps:

- Step1 : Run ```npm install``` it will install all dependencies
- Step2 : Rename ```.env.example```  to ```.env``` it will set all environment variables
- Step3 : Run ```npm run dev``` to run in development mode
